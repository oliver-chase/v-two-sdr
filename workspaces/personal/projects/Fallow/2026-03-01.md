## How It Works End-to-End

1. **User opens app** → React loads from http://localhost:3000
2. **UI fetches events** → GET /api/events returns canonical events
3. **User clicks "Run Sweep"** → POST /api/sweep triggers unified-sweep.js
4. **Scheduler runs** → Automatically runs sweeps every 24h (configurable)
5. **Pipeline executes** → Consolidates, matches, deduplicates events
6. **Results saved** → matched_events.json + sweep logs
7. **API updated** → /api/status + /api/matches return live data
8. **UI refreshes** → StatusPanel shows new metrics, matched count updates
